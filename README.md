# Velox Living

Presentation website for Velox Living — property management services in Târgu Mureș, Transylvania.

## Structure

- `index.html` — Landing page
- `properties/` — Individual property pages (property-1, 2, 3)
- `css/` — Stylesheets
- `js/` — JavaScript modules
- `i18n/` — Translation files (EN, RO, HU, ES)
- `assets/` — Logo, icons, images

## Languages

English | Română | Magyar | Español

## Deployment

Deployed via GitHub Pages. Push to `main` triggers auto-deploy.
